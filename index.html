🚀 Convertir un site en APK Android

Entre l'URL de ton site web (il doit avoir un manifest.json valide)

``` --- ## 🔑 Les 2 choses à remplacer dans `index.html` | Ce que tu vois | Ce que tu mets | |---|---| | `TON_USERNAME` | Ton pseudo GitHub (ex: `jean123`) | | `TON_TOKEN_GITHUB` | Un token GitHub (voir ci-dessous) | --- ## 🔐 Créer ton token GitHub (1 minute) 1. Va sur → **github.com > Settings > Developer settings > Personal access tokens > Tokens (classic)** 2. Clique **"Generate new token"** 3. Coche uniquement : ✅ `repo` 4. Copie le token et colle-le dans `index.html` --- ## ▶️ Comment ça marche ensuite ``` Tu ouvres index.html dans ton navigateur ↓ Tu entres l'URL de ton site PWA ↓ GitHub Actions se déclenche automatiquement ↓ GitHub > ton repo > onglet "Actions" > télécharge l'APK
